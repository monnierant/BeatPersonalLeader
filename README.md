# BeatPersonalLeader

A web app to view your **Beat Saber** stats and scores from [BeatLeader](https://beatleader.com) — including **all songs**, not just ranked ones.

Enter any BeatLeader profile URL or player ID to browse scores, accuracy, and stats with filtering, sorting, and charts.

## Tech Stack

React · TypeScript · Vite · Redux Toolkit · Styled Components · Recharts · GitHub Pages

## Getting Started

```bash
npm install
npm run dev
```
